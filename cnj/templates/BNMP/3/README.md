# BNMP

## Serviços da Stack Atual
- Cliente
- Serviço
- Alertas
- Gateway
- Gateway Externo
- Elasticsearch
- Banco para Serviço (opcional)
- Banco para Gateway (opcional)

- Habilitação do volume via driver **NFS**.
- Habilitação de criação dos bancos de dados.

- bnmp-pgsql-gateway
